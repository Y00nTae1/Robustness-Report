# 단계별 실행 카드 v2 — 비전 보정 폐루프 강건성 검증 과제

`playbook_v2.md`의 프롬프트를 **빈칸 없이 채운** 실행 카드 모음. 각 카드를 통째로 복사해 Claude Code에 입력한다.

## v2 반영 사항
- 모든 P2 카드에 **[필수] 실제 실행 + 로그·산출물 첨부** 항목.
- 모든 배경에 **비ASCII(한글) 경로 우회** 명시.
- S2~S6의 P1 카드에 **S1 확정 사실(부록 B)** 을 직전 산출물 요약으로 포함.
- 모든 P3 카드에 **"단계에 없는 요구로 감점 금지"** + **"실행 증거 없으면 실행/산출물 1점 이하"**.
- S1 평가표 "부품 → 로봇 가시성" 정정 반영.

## 사용법
1. 단계는 S1 → S7 순서. 한 단계 안에서 P1 → P2 → P3 → P4 순서로 카드를 하나씩 입력.
2. **한 번에 여러 카드를 붙이지 마라.** 한 카드 입력 → 출력 확인 → 다음 카드.
3. `<<< 직전 출력 붙여넣기 >>>` 자리에 바로 앞 프롬프트의 출력을 복사해 넣는다.
4. P4 통과 시 → 다음 단계 P1로(산출물 요약 전달). 불합격 & 3회 미만 → 수정 지시를 P2 피드백에. 3회 이상 → 한계 기록 후 다음 단계로.

> S1은 이미 통과(✅). S2-P1도 실행 완료. 이 문서는 **S2-P2부터** 바로 이어서 쓸 수 있다. (S1 카드는 참조용으로 유지)
